STARTING A PROJECT
npm init -y
npm install express --save
npm install ejs --save
npm install body-parser --save
npm install socket.io --save
npm install mongoose --save

***IN ONE LINE***
npm init -y && npm install express --save && npm install ejs --save && npm install body-parser --save && npm install socket.io --save && npm install mongoose --save


**OPTIONAL MODULES**
npm install mongoose-validator --save

INSTALL NPM PACKAGES
npm init -y
**copy dependences from project into package.json**
npm install

ADD TO SERVER.JS
var express = require("express");
var bodyParser = require('body-parser');

var app = express();

app.use(bodyParser.urlencoded({extended: true}));

SET VIEWS DIRECTORY
app.set('views', __dirname + '/views');


SETTING ROUTES
app.HTTP_VERB('URL', function (req, res){

});

REQ KEY WORDS
req.body **POST DATA SENT FROM FORMS**
req.params **DATA SENT THROUGH URL**

RUNNING A SERVER
nodemon server.js


TO USE SESSION DATA
npm install express-session
